TwilioLookup

public class TwilioLookup

Class used to lookup phone number using the Lookup feature of Twilio.

To use TwilioLookup methods you need to set the accountSid and accountToken found in the Twilio console dashboard (Twilio dashboard).

    type

    Indicates the type of information you would like returned with your request. Possible values are carrier or caller-name. If not specified, the default is nil. This properties is available only on paid accounts.
