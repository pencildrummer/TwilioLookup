TwilioLookup

public class TwilioLookup

Class used to lookup phone number using the Lookup feature of Twilio.

To use TwilioLookup methods you need to set the accountSid and accountToken found in the Twilio console dashboard [https://www.twilio.com/console/sms/dashboard].

  • lookup(_:countryCode:type:completion:)
